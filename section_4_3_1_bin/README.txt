2. Unzip the contents of the zip file into a local folder on your machine,let us call it the 'app' folder. The folder would contain the following files :
	-CODAPOS.jar
	-pos.data
	-README.txt


Setting up Required Libraries
-----------------------------

The following libraries are required for building and executing the code

1. db4objects Database : Please download db4objects database from http://www.db4o.com/DownloadNow.aspx

2. Extract the content of the downloaded zip to a folder ,let us say 'db4o'.

3. Browse to the lib folder inside the 'db4o' folder.Typically it would be something like db4o\db4o-8.0-java\db4o-8.0\lib.

4. Copy a file similar to 'db4o-8.0.249.16098-all-java5.jar' from 'db4o\db4o-8.0-java\db4o-8.0\lib' to the 'app' folder.


Running the program
-------------------

1. The program can be run from your OS console window provided Java Runtime Environment is installed and configured. 

2. From the command prompt, navigate to the 'app' folder. 

3. Type the command 'java -cp .;./* pos.app.POSAPP' to start execution of the program which would present a text based menu.

4. Use the POS program as guided by the text based menu titled 'Welcome to Restaurant'.
